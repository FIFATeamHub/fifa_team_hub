## Rituais da Equipa

| Ritual | Frequência | Canal | Objetivo |
|--------|-----------|-------|----------|
| Reunião de Sprint | Toda quinta-feira | Discord | Retrospectiva + planejamento da próxima sprint |
| Prazo de issues | Sábado 12h | GitHub Issues | Entrega das tarefas da sprint |
| Code Review | A cada PR | GitHub | Qualidade, segurança, isolamento |
| CI/CD | A cada PR | GitHub Actions | Testes + lint bloqueadores |
